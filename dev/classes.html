API Reference¶

This is the class and function reference of grakel. .. Please refer to the full user guide for further details, as the class and .. function raw specifications may not be enough to give full guidelines on their .. uses. .. For reference on concepts repeated across the API, see Glossary.

grakel.graph: Graph class with its utility functions¶

Base Class¶

grakel.Graph([initialization_object, …]) The general graph class.

Utility Functions¶

grakel.graph.is_adjacency(g[, transform]) Define if input is in a valid adjacency matrix format.
grakel.graph.is_edge_dictionary(g[, transform]) Define if input is in a valid edge dictionary format.
grakel.graph.laplacian(csgraph[, normed, …]) Return the Laplacian matrix of a directed graph.
grakel.graph.floyd_warshall(adjacency_matrix) Calculate the Floyd Warshall, shortest path matrix.

User guide: See the Graph (class) section for further details.
